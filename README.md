## Tech Stack

- **HTML5**
- **CSS3** (modular approach with separate files)
- **Vanilla JavaScript**

## Features (Planned / Implemented)

- ✅ Responsive layout
- ✅ Modular CSS structure (`reset`, `header`, `footer`, `home`, etc.)
- ✅ Clean and airy modern design
- 🔲 Dynamic project loading (JSON)
- 🔲 Individual project detail pages
- 🔲 Solutions overview page
- 🔲 Contact form with validation
- 🔲 Site map and credits page

## How to Run the Project

1. Clone or download the repository
2. Open the `html/index.html` file in any modern web browser (Chrome, Edge, Firefox recommended)
3. The website will run locally without any server or installation required

## Design Goals

- Light neutral color palette with soft blue/teal accents
- Generous whitespace for an airy and optimistic feel
- Consistent navigation and layout across all pages
- Professional yet friendly tech company appearance
